# computers
Prosta strona napisana w Laravelu, na której można utworzyć komputer wraz z jego specyfikacją. Na ten moment strona ma bardzo prostą funkcjonalność ograniczoną do utworzenia prostegoi profilu komputera z podstawową specyfikacją, jednak w przyszłości strona zostanie rozbudowana o dodatkowe opcje (np. możliwość rejestracji, dodania zdjęcia komputera, większej ilości podzespołów, ulepszenia szaty graficznej itp.)
